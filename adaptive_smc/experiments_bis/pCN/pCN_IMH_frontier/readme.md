This is the experiment file to recover the critical regime for pCN (Sect. 5.1)
The two target distributions are a standard scaled Gaussian distribution, and a Gaussian distribution with a diagonal
covariance matrix with entries vanishing to $0$ like $1/k^{\alpha}$, $\alpha\geq 1$. The critical temperature for the
first
is $\lambda_{\textup{C}}\propto d^{-1}$, whereas for the second it is $\lambda_{\textup{C}}\propto d^{\alpha-1}$,
for $\alpha=1$, $\lambda_{\textup{C}}\propto 1/ \ln d$.
